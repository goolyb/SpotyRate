Где взять CSV своего плейлиста

Spotify — через Exportify (даёт аудио-фичи для полной оценки):

  1. Открой exportify.net и нажми Get Started / Log in with Spotify.
  2. Разреши доступ — Exportify покажет список твоих плейлистов.
  3. Напротив нужного плейлиста нажми кнопку Export (значок скачивания).
  4. Скачается .csv файл — его и загрузи сюда выше. 🎉

Apple Music, YouTube Music, SoundCloud и другие — через конвертер плейлистов:

  1. Открой Soundiiz или TuneMyMusic и подключи свой сервис.
  2. Выбери плейлист и в качестве назначения укажи экспорт в CSV / файл.
  3. Скачай .csv и загрузи его сюда. Формат определится автоматически.
Подойдёт и нативный экспорт: YouTube Music — через Google Takeout; Apple Music — в приложении «Музыка»: Файл → Медиатека → Экспортировать плейлист…
У всех сервисов кроме Spotify нет аудио-фич — оценка считается по популярности и разнообразию.
Ничего никуда не отправляется: файл читается прямо в твоём браузере.
